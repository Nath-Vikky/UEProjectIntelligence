# UEPI Terminology

- Entity: A stable node in the project knowledge graph.
- Relation: A typed directed edge between two entities.
- Evidence: A source pointer explaining why an entity or relation exists.
- Snapshot: A structured JSON document attached to an entity.
- Artifact: A file-backed payload for large or range-readable data.
- Completeness: Explicit coverage, omissions, warnings, and state for an extraction result.
- Source layer: The confidence/provenance tier for extracted data.
- Canonical graph: Pin-level source graph.
- Projection graph: Derived node-level graph such as CFG, DFG, call graph, or asset reference graph.
- Revision: Historical asset state recorded on ingest.
