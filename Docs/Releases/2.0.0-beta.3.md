# UEProjectIntelligence 2.0.0-beta.3

This beta hardens the high-frequency read path found during the 2026-07-17 LLMNPCDemo recheck.

## Highlights

- Generation-addressed SQLite caches avoid replacing a database held by another Windows reader.
- MCP requests explicitly close query engines and SQLite connections.
- Cache sync failures surface as `UEPI_CACHE_SYNC_FAILED` while reads fall back to Snapshot fragments.
- Incremental event freshness reads parse a bounded file tail instead of rereading the full history.
- Blueprint filters correctly unwrap typed attributes and distinguish a real filter miss from missing graph data.
- Hard-scoped animation context preserves every root asset and selects explicitly named animation sequences before AnimBP child entities.
- `uepi_edit_discover` now declares projection, payload-budget, and pagination parameters and emits a complete next-page action.
- Status and Doctor identify `plugin_version`, `plugin_build_id`, `catalog_hash`, and `service_version`.
