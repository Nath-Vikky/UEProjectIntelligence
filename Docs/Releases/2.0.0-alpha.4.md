# UEPI 2.0.0-alpha.4 Verification

Date: 2026-07-12

## Scope

This experimental release hardens Editor startup, exact Blueprint scope, stable Blueprint pin schemas, touched-only transaction scope, and transaction diffs. It adds a passing UE5.3.2 Third Person Golden record while remaining Alpha until the rest of the real-machine matrix passes.

## Verified

- `GasDemoEditor Win64 Development` builds with UE 5.3.2.
- `python -B Tools/test_snapshot_mcp_v2.py` passes, including duplicate target/dependency and transaction-diff regressions.
- Online GasDemo Doctor passes 17/17 checks.
- All 64 Operation Registry entries are apply-supported under the enabled GasDemo gates.
- One approved transaction completed duplicate, Blueprint variable/node/connection edits, compile, save, targeted refresh, semantic diff, UEPI-owned PIE start/stop, Editor restart, and offline readback.
- Source Blueprint bytes remained unchanged; the Golden package persisted with the exact recorded SHA-256.
- A runtime ticket bound to the old Editor session was rejected after restart.
- Source and UE5.3.2 Win64 archives use one root, carry matching descriptor metadata, exclude generated/private directories, and pass content inspection.
- The extracted Source archive passes UE5.3.2 `RunUAT BuildPlugin -TargetPlatforms=Win64 -Rocket` in an isolated host project.
