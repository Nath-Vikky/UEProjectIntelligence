# UEPI 2.0.0-alpha.3 Verification

Date: 2026-07-12

## Scope

This experimental release completes the structural migration of every public write operation to concrete Operation Registry handlers. It remains Alpha until the UE5.3.2 real-machine matrix passes.

## Verified

- `GasDemoEditor Win64 Development` builds with UE 5.3.2.
- `python -B Tools/test_snapshot_mcp_v2.py` passes.
- `python -B -m compileall -q Services/uepi/src/uepi Tools` passes.
- Offline Doctor passes all required checks for GasDemo; Editor-only checks are warnings while the Editor is closed.
- Source and UE5.3.2 Win64 release archives use one root directory and matching `2.0.0-alpha.3` descriptor metadata.
- Source archive excludes `Binaries`, `Saved`, `Intermediate`, and `__pycache__`.
- Prebuilt archive includes only the expected Win64 plugin binaries in addition to source-release files.
- The extracted source archive passes UE5.3.2 `RunUAT BuildPlugin -TargetPlatforms=Win64 -Rocket` in an isolated directory.
- The packaging tool rejects a requested version that differs from `UEProjectIntelligence.uplugin`.
